**************TURTLES WAR ZONE***************

A game based on the turtles predefined moves given in the problem file of the repository.Project is totally implemented using python language and a well defined interface is implemented using pygame library of python.

A human vs Bot game(Agent)
 

Prerequisites:

1. Python 2.7
2. PyGame library

Steps to Run the game in your pc:

  a. Clone the directory as it is.
 
  b. Run the command in your terminal: 
      > python main.py

  c. The game starts,enjoy playing.
